# Weather App 🌤️

A simple Python GUI app to check the current weather in any city. Built with Tkinter, Requests, and Pillow.

Features
Enter a city and get current weather 🌦️
Shows temperature in Celsius and weather description
Displays weather icon 🌞🌧️
Installation
git clone https://github.com/YOUR_USERNAME/WeatherApp.git
cd WeatherApp
pip install -r requirements.txt
Add your WeatherAPI key in config.py:

API_KEY = "YOUR_API_KEY"
